We'll be using UCROO (www.ucroo.com) for class collaboration this Semester.

UCROO saves time and improves teaching performance, while providing students with an innovative and engaging tool for an enriched learning experience.

They support linking of videos, embedding of images, and attaching of files, as well as a number of other useful teaching tools including endorsing of answers, assessment reminders and voting.
